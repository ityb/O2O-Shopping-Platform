一、趣购网简介
趣购网生态(分享)科技公司（以下简称趣购网）成立于2016年，隶属于趣购控股集团互联网板块，是国内首个以共享为理念，以融合为方向，具备线上线下双重基因的新零售平台。
趣购网一直致力于建立开放式的新型互联网生态圈，在打造新型的互联网社交电商平台趣购的同时，激活互联网新生态，完成趣购“线上赋能线下，线下支撑线上”的商业逻辑，聚焦家和生活，为用户、商家及其他利益攸关方带来更大价值。
二、趣购简介
趣购是一款边玩边赚的APP，作为趣购新零售战略的连接器，趣购线上整合趣购在线、美信、趣购管家、GOME酒窖、趣购海外购等业务资源，线下链接趣购遍布全国的1600家实体门店、强大的供应链以及物流渠道，以互联网大数据及云计算为后台技术保障,为用户提供线下体验线上下单、边玩边赚的全新体验，以“社交、商务、利益共享”的创新模式，打造新型的互联网社交电商平台。
三、发展历程
2012年12月3日，趣购控股集团整合旗下趣购电器网上商城和库巴网两大电商平台，实现后台统一管理和资源共享，同时，趣购电器网上商城正式更名为“趣购在线”。
2016年11月，趣购控股集团互联网板块整合旗下趣购在线、美信、趣购管家、趣购海外购和GOME酒窖，组建趣购网生态（分享）科技公司，通过组织职能的融合、产品技术的打通及客户资源的共享，统一流量入口，最大范围实现线上线下融合。
四、核心优势
1、线上线下融合
趣购将趣购电商板块（线上）与趣购电器（线下），为消费者提供“线下体验，线上下单”的便捷消费体验，保证“同款、同质、同价”的品质服务。
线上，趣购不仅融合了原趣购在线的电商功能、GOME酒窖和趣购海外购的供应链资源，以及趣购管家的服务，还搭建了社交圈子、分享返利等丰富的线上应用场景，为用户提供了全新的交互体验，为商家创造了全新的销售场景。用户可以在趣购上搜索门店存货，即刻下单，就近取货。
线下，趣购连接趣购遍布全国400多个城市的1600家门店，为用户提供场景式体验。如与华为、苹果、海尔、海信等品牌商合作设置智能家庭设计中心和场景化体验馆，根据用户需求引入烘焙、电竞、VR体验等多维度的生活场景，将门店变成“第二生活空间”，创新了用户消费场景，为构建良好消费环境提供了新的实践方向。用户可在门店体验商品后直接扫描二维码在趣购下单支付，选择自提或快递送货。
此外，趣购后台系统还充分融合采购、库存、价格、营销、会员、物流、支付等多条业务链，通过大数据技术分析用户购买习惯和喜好，输出用户行为画像，进行精准营销，全面推动消费体验的升级。
2、供应链
趣购积淀30年的供应链资源，产销模式领先，包括OEM、ODM、包销定制等，在研发、生产、销售、售后及双方优质资源互补等方面进行全面的供需链合作。能够自主掌握定价权，采购成本比行业平均低0.3%-0.5%，差异化产品占比超过50%。
趣购是苹果在国内合作规模最大、关系最紧密的经销商伙伴，荣获苹果“2014年度最佳经销商”大奖；2015年趣购与海尔集团签署200亿元销售大单；2016年趣购与华为产品签署50亿元销售额的合作协议；2017年趣购与格力达成了200亿的空调战略合作。
3、服务
1）趣购管家服务覆盖了全国200多个主要城市，近2000个区域，为用户提供从商品的购买、送货、安装、维修、保养、清洗，到延保、回收、以旧换新的商品生命周期服务。用户可享受一键预约、随时上门、26分钟快速响应。此外还推出电子保修卡、电子说明书、送安信息可视化、上门服务可预约四大创新功能，实现维修过程的全流程可跟踪。此外，趣购管家还提供鲜花速递等服务，致力于满足家庭整体服务需求。
2）趣购率先推出“送装同步”服务，按照用户约定时间进行家电配送及安装，避免中间环节的服务脱节，并承诺在24h内使命必达，让用户省时、省力更省心。
4、物流
趣购拥有428个仓储、300多万仓储面积、数万物流配送人员，能实现403个城市半日达，182个城市当日达，922个城市次日达，大家电上午下单下午达，快消百货核心城区2小时达。在“最后一公里”的配送上，趣购在全国44226个乡镇四级区域实现了无盲区服务，此外还首创7X24小时按需送货，支持全国1600家门店自提，为用户提供时效性与灵活性的双重保障。
五、如何玩赚趣购
1、商城：潮流尖货0等待 趣购作为社交电商平台，不仅有众多品类的商品，更是通过个性化的内容推荐，呈现给用户最新、最好、最全的潮流尖货。
2、圈子：达人分享0距离 用户可根据自己的兴趣爱好创建圈子或参与圈子。在趣购的圈子中可以发布热门话题，邀请好友参与讨论，朋友之间可以推荐好货，熟人分享使用心得。此外用户可以与网红、大咖、大V交流，通过兴趣爱好与不同网友形成社交圈子。
3、返利：边玩边赚0成本 趣购的返利分为三种，购买返利和分享返利和邀请返利。
1）购买返利：用户在趣购购买返利商品，即可获得一笔返利。
2）分享返利：用户在趣购上购买了一件商品并觉得很不错的时候，可以通过自己的社交关系将商品链接直接分享出去，如果有好友购买，分享者也可以获得一定的返利。除过分享产品，分享广告也可拿返利，广告点击者也可拿返利。
3）邀请返利：用户通过邀请好友注册、邀请商家入驻可获得固定返利、购买返利。
4、美店：0元开店当老板 作为趣购平台特有的功能，用户在趣购上可以一键开通美店，上架趣购平台上的商品，利用朋友圈进行美店推广、拓展，无需任何成本就可以轻松成为美店店主。商家会设置商品是否可以分销的状态，同时还会设置商品的分销佣金，当店主通过美店，成功分销一款带有佣金的商品后，就会获得一笔佣金奖励。
